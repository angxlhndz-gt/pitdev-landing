# PitDev Landing Page

Landing page profesional para PitDev, creada con React, Vite y Tailwind CSS.

## Requisitos

- Node.js 20 o superior
- npm

## Instalación

```bash
npm install
```

## Ejecutar en desarrollo

```bash
npm run dev
```

Luego abre la URL que muestre Vite en la terminal.

## Compilar para producción

```bash
npm run build
```

## Configuración

Los datos principales de la marca, servicios y redes sociales están en:

```text
src/config/site.js
```

Desde ahí puedes actualizar enlaces de WhatsApp, Instagram, Facebook, servicios y textos principales sin repetirlos por toda la página.
